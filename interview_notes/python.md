# Python Interview Notes

## Logging

### What is Logging?
Logging is the process of recording application events.

### Why do we use it?
To monitor and troubleshoot applications.

### Where is it used?
ETL pipelines, Airflow DAGs, Cloud Functions, Dataproc, Dataflow.

### When should we use it?
In production applications.

### Real-time Example
Logging ETL start, file read success, and BigQuery load failures.
